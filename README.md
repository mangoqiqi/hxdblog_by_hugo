# hxdblog_by_hugo
这里是一个hugo博客，想要安装在你的机器上，你必须先安装docker，或者dockertoolbox。
![blog](http://pbqgh436d.bkt.clouddn.com/18-7-15/49270266.jpg)
**Usega**
```
$ git clone git@github.com:mangoqiqi/hxdblog_by_hugo.git
$ cd hxdblog_by_hugo
```
**Build**
```
$ sh bilud.sh
```
**Run**
```
sh run.sh
```
**Stop**
```
$ sh stop.sh
```
